# API de Pedidos

Projeto de teste técnico para praticar construção de API REST.

## Tecnologias utilizadas

- Node.js
- Express
- Swagger
- Git
- GitHub

## Estrutura do projeto

src/
controllers/
routes/
services/
utils/

## Endpoints

POST /order - Criar pedido

GET /order/:id - Buscar pedido

PUT /order/:id - Atualizar pedido

DELETE /order/:id - Remover pedido

## Executar projeto

Instalar dependências

npm install

Rodar servidor

node index.js

Servidor roda em:

http://localhost:3000

## Documentação da API

Após rodar o projeto acesse:

http://localhost:3000/api-docs
